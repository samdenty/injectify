## `windowspam` module

Runs the window.open command in a loop (thats a lot of windows)

## Usage

```js
injectify.module('windowspam')
// undefined


injectify.module('windowspam', 'https://samdd.me')
// undefined


injectify.module('windowspam', {
    url: 'https://samdd.me'
})
// undefined
```
